# PairPaoint 

### A Node.js & Express API Project

## Project Overview
**PairPoint** is a backend API developed using **Node.js** and **Express.js**.  
The project focuses on building scalable, RESTful APIs with proper routing, middleware usage, JWT and clean architecture.  

## Tech Stack
- **Node.js**
- **Express.js**
- JavaScript (ES6)
- REST API
- JSON
- Postman (for API testing)

## Features
- RESTful API structure  
- Express routing & middleware  
- Request & response handling 
- Database Integration 
- User Authentication (JWT)
- Role-Based Access Control
- Error handling (404, server errors)  


## Installation & Setup

1. Clone the repository
```bash
    git clone https://github.com/your-username/pairpoint.git

2. Navigate to the project directory
    cd pairpoint

3. Install dependencies
    npm install

4. Start the server
    node app.js
or (if using nodemon)
    nodemon app.js
